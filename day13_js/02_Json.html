JSON(javascript Object Notation) : 텍스트 데이터를 사용하여 데이터를 저장하고, 전송, 데이터 공유를 위한 개방형 표쥰 파일 형식

  • 데이터 구조화 : 많은양의 데이터가 실시간으로 전달되어야 할 때 가장 효율적인 방식
  • 구조화된 텍스트를 만들기 위한 JSON 데이터를 많이 활용하고 있음
  • 자바 스키립트의 객체 형식을 갖고 있기 때문에 구조 파악이 쉽고, 제작도 쉽다
  • 웹 분야의 자바스크립트가 모든 프론트에 호환이 된다는 점을 활용
  • 데이터 수집, 출력용으로 거의 모든 플랫폼 서비스에서 사용되고 있음
